Conectar usuario

En base a la api de conectar usuarios se obtienen los valores del usuario.

Añadir plan de comidas

En base a la api de añadir plan de comidas se pueden agregar recetas, ingredientes o un plan por comida al plan diario.

Generar Plan de comidas

En base a la api de generar plan de comidas y basados en los parametros ingresados por el usuario se obtiene un plan de comidas de un dia.

Obtener plan de comida diaria

En base a la api de obtener plan de comidas y basados en los parametros ingresados por el usuario se obtiene el plan de comidas de un dia.

Eliminar plan de comida diaria

En base a la api de eliminar plan de comidas y basados en los parametros ingresados por el usuario se puede eliminar el plan de comidas de un dia.
